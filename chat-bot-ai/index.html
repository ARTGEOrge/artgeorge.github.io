Modern AI, running in your browser

This page loads an open-source LLM directly into your browser using WebGPU — no server, no API key, your messages never leave your device.

The first message will trigger a one-time model download (a few hundred MB). It's cached after that.

Live headlines from WSJ, FT, BBC, and CNN + a Wikipedia lookup are pulled in for every question — so it can answer about specific people, events, and elections, not just recent news.
