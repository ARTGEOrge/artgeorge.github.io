Say things

say "Hello!"
say 2 + 2
say "Score:", score

Variables

set name to "Art"
set score to 10
set score to score + 5

Decisions

when score > 100 {
  say "You win!"
} otherwise when score > 50 {
  say "Almost!"
} otherwise {
  say "Keep going"
}

Compare with =, is, !=, is not, < > <= >=. Combine with and, or, not.

Loops

repeat 3 times {
  say "hi"
}

repeat with i from 1 to 5 {
  say i
}

while score < 3 {
  set score to score + 1
}

Functions

define greet(who) {
  give "Hello, " + who
}
say greet("world")

give returns a value.

Lists

set xs to [10, 20, 30]
say xs[0]
add(xs, 40)
say length(xs)
