🔗 Turn on sync

Sync saves your files to a private GitHub repository (artdrive-storage) so ArtDrive shows the same files on every computer you connect.

  1. Create a GitHub token (the repo box is pre-checked — scroll down, click Generate token).
  2. Copy the token and paste it below.
  3. Do the same on your other laptop with the same token.

The token is saved only in this browser. Anyone with it can use your GitHub — don't share it.
