“Play One Tournament” simulates a single random run from groups to the final.

How it works. Each team has an Elo strength rating (estimated for early 2026 form / FIFA ranking). Matches are simulated with a Poisson goal model derived from the Elo gap; the three hosts get a small home-field bump. Group standings use points → goal difference → goals scored, the eight best third-placed teams advance, and a seeded 32-team knockout bracket plays out (ties decided by a rating-weighted shootout). Run thousands of tournaments and the win-% column is how often each nation lifted the trophy. This is a toy model for fun — not betting advice.
